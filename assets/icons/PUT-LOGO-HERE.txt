Official BOORAY logo import slot
================================

This file marks where your pixel-perfect logo goes.

CREATE or REPLACE this file (same folder):

  assets/icons/booray-logo-source.png

Requirements:
  - PNG (your official artwork)
  - Square is best; script normalizes to 1024x1024 on black
  - Opaque background (no transparency for App Store)

Then run:

  npm run icons:generate

Or import from anywhere on your machine:

  npm run icons:import -- /path/to/your/booray-logo.png

Delete this PUT-LOGO-HERE.txt after you add the PNG (optional).
